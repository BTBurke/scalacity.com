A high performance architecture without the headaches
  • Key features:
    • Define complex application architectures with a simple text file
    • Specify dependencies and configuration items that are automatically provisioned to your nodes using Puppet
    • Define scaling rules that respond to custom application or machine metrics
    • Easily integrate with your existing data stores and cloud architecture
  • Scalacity is designed to solve your devops headaches when developing complicated applications.

    How does it work?

    Define your application's architecture, its configuration, and dependencies in a simple text-based format. We'll automatically analyze your application and your custom scaling rules to determine how best to deploy your code to your cloud provider. Scalacity integrates with your existing data stores and cloud architecture to launch, provision, and monitor your application.

    Deploy Storm, Spark, or Hadoop applications easily without worrying about configuration headaches. Or, develop your own application framework with fine-grained control over all aspects of your configuration. Start from our battle-tested Puppet configurations and fine-tune them as needed.
Insight into every facet of your application
  • We help you get the best possible performance by giving you deep insight into what's going on with your application.

    How does it work?

    Visualize every important metric in your application. Quickly identify bottlenecks created by machine capacity, the network, or define custom application metrics to understand where your code is slow. Our web interface allows you to drill down so you can see when things aren't working and why.

    Your metrics and logs are fully indexed and searchable via the web interface or through our API. Run different versions of your application and compare performance metrics, or go back in time and see how and why your application's performance has changed.
  • Key features:
    • Visualize all the important metrics in your application
    • Use the web interface to drill down on any facet of your application and monitor its performance
    • Metrics and logs are fully indexed and searchable
    • Integrate metrics with custom rules like service notifications or scaling rules
